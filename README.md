<hr />

## 스크립트

프로젝트 폴더에서 다음을 실행할 수 있습니다.

### `npm start`

개발모드로 리액트를 실행힙니다.<br />
실행된 인스턴스를 [http://localhost:3000](http://localhost:3000)에서 확인하실 수 있습니다.

수정이 일어날때마다 페이지는 새로고침됩니다.<br />
또한 에러를 콘솔에서 확인할 수 있습니다.

### `npm run build`

배포를 위해 해시를포함한 압축된 파일을 생성(빌드)합니다. `build` 폴더에 생성됩니다.
